# Cinema
Projeto desenvolvido na linguagem Java com interface gráfica e arraylist, elaborado para a disciplina de Programação Orientada a Objetos. 
Consiste um sistema no qual o operador cadastra o filme, pode atualizar as informações sobre esse e excluir o filme, conforme o arraylist for alimentado é exibido na interface gráfica um tabela com todas as informações sobre o filme.
